Instalando Bootstrap

No site do Bootstrap copiar o código e colar: CSS acima de title, e JS antes de fechamento body.

Para adicionar CSS e JS de remoto para local: pegar apenas o caminho do link, exemplo: "https://cdn.jsdelivr.net/npm/bootstrap@5.2.0-beta1/dist/css/bootstrap.min.css" abrir no navegador, comando ctrl+s, salvar na pasta correspondente, no caso CSS.

Após organizado os links em suas devidas pastas, colocar link css , abaixo do title e chamar a pasta e arquivo, e para o JS chamar script src, e também, pasta e arquivo correspondete.
